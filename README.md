# playlistbot_gui
Simple GUI for playlist bot


Compatibility: `Windows`, `Linux`

# Installation
```
git clone https://github.com/0bliss/playlistbot_gui
cd playlistbot_gui
python3 -m pip install -r requirements.txt
```

# Usage
```
python3 main.py
```
